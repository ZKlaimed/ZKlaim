### Technical Details

1. **Coverage Selection:** User picks from available coverage types
2. **Configuration:** Parameters stored locally until proof generation
3. **Attestation:** Required for non-parametric products (auto, health)
4. **Proof Generation:**
   - WASM module loaded (~5MB, cached)
   - Private inputs processed locally
   - Proof generated in 10-60 seconds
5. **Transaction:** Proof + premium submitted to Aleo
6. **Confirmation:** Policy record created on-chain

### Technical Details

1. **Pool Discovery:** User browses available pools with stats
2. **Risk Assessment:** View historical performance, loss ratios
3. **Deposit:** USDC transferred, LP tokens minted
4. **Yield Accrual:** Premiums distributed to LP holders
5. **Withdrawal:** LP tokens burned, USDC returned
6. **Constraints:** Withdrawal limited by reserve requirements
